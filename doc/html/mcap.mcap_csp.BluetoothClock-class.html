read(self, piconet)

source code 

Reads Bluetooth clock.

If piconet is True, reads piconet clock. If False, reads native clock (which is the same as piconet clock if local BT is piconet master).

Returns a tuple with BT clock and accuracy, or None if not successful.

Accuracy may be zero (means theoretical infinit precision). Unit is Bluetooth "ticks" (312.5 us each), wraps 32 bits
